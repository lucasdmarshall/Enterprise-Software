# Frontend

Client-facing applications and shared UI packages.

Suggested layout:
- `apps/pos/` – POS terminal UI.
- `apps/admin/` – Admin/Back-office UI.
- `packages/ui/` – Shared UI components.
- `packages/shared/` – Shared frontend utilities/types.

Add your framework of choice (e.g., React/Next, Vue/Nuxt, SvelteKit).
